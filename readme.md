## Sum of Numbers

### This package is used to find the sum of consecutive n numbers

- To install the package

```js
npm i @aiswarya_r/sum_numbers
```
- To use the package

```js
const sum = require("@aiswarya_r/sum_numbers");
console.log(sum(5));
//15

```
